# ReadHelper 阅读助手

一个用于 Tampermonkey 的油猴脚本，帮助把任意网页切换为更适合长时间阅读的配色主题。

## 主要功能

- 19 套内置主题：浅色护眼、冷灰编辑器、深色 IDE 风、暖炭深色等，支持办公/代码/夜间多场景。
- 一键切换：点击左下角圆形按钮即可循环切换主题。
- 设置面板：点击齿轮按钮打开面板，可切换主题、设置自定义背景图片及透明度。
- 广泛覆盖：对 Ant Design、Element UI、iView 等常见组件库的表格、表单、标签页、下拉菜单、弹窗等进行颜色覆盖。
- 广告过滤：自动隐藏常见浮动广告元素，减少视觉干扰。

## 安装使用

1. 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展。
2. 在 Tampermonkey 中新建脚本，将 `ReadHelper.user.js` 的内容完整粘贴进去并保存。
3. 打开任意网页，左下角会出现悬浮按钮，点击即可切换主题。
